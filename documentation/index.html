Overview

This is a HTML5, responsive template for Construction, Corporate farm and Web Agencies based on Twitter Bootstrap 4

HTML Structure

Creative comes with few styles for Sections (including footer) with 100% and container, container-fluid classes are used to limit width of the section

<section class="section">
   <div class="container">
   <!--and innter tags, divs-->
   </div>
</section>

Another example with "seaction-header"

<section class="section">
   <div class="container">

       <div class="section-header">
           <!--section header content here-->
       </div>
      
      <!--section body content here-->
   </div>
</section>

Data Attributes

Besides Bootstrap's data attributes you may see some foreign attrubutes

data-src

See on "Lazy Load" section

data-wow-delay, data-wow-duration

Together with "wow" class they are used to create appear/reveal animation. For more information and option, pls check WOW JS's documenation (link at the Scripts & Resources section).

data-speed, data-to, data-from

JQuery countTo plugin uses these attributes to control count speed, count to and count from value. For more information, pls check counTo's documenation (link at the Scripts & Resources section).

 

 

Slider & Carousel

Owl Carousel2 is used in the slider section and for "Reviews", Bootstrap's carousel is used

(Main) Slider with Owl Carousel2

Asset files included

  • owl.carousel.min.css
  • owl.theme.default.min.css
  • owl.carousel.min.js
  • animate.css
  • custom-animations.min.css (containing some custom amiation for slide text)

HTML

Its pretty simple

<div id="slides" class="owl-carousel owl-theme">

   <div class="slide owl-lazy" data-src="image.jpg">
      <div class="content classess....">
            <!--slide text here-->
      </div>
   </div>

   <!--slide 2-->
   <!--slide 3-->
</div>

JS

OwlCarousel is intiated in the footer js files. If you're using uncompressed js files, it should be in int.js

$('#slides').owlCarousel({
    animateIn: 'fadeIn',
    animateOut: 'fadeOut',
    items:1,
    lazyLoad:true,
    dots:false,
    loop:true,
    autoplay: true,
    autoplayTimeout:10000,
    .....
});

OwlCarousel can use Animate.css (link provided at the resources section) animations for slide in and out transitions. For example:

animateIn: 'fadeIn',
animateOut: 'bounceOut',

Custom animations for slide text

Classes that are used in slide text

  • slideLeftIn
  • slideRightIn
  • slideBottomIn1
  • slideBottomIn2
  • slideBottomIn3
If you're changing slider's autoplayTimeout, adjust animation-duration in custom-animation.css

Portfolio/Facilities

Portfolio section uses Isotope plugin and required asset file is-

isotope.pkgd.min.js
imagesloaded.pkgd.min.js

 

This section has mostly 2 parts - filters and items.

Filters

As the title suggests, filters do filter portfolio items.

<div class="portfolio-filter text-center">
   <button class="btn btn-active" data-filter="*">All</button>
   <button class="btn btn-link" data-filter=".tag1">Tag1</button>
   <button class="btn btn-link" data-filter=".tag2">Tag2</button>
   <button class="btn btn-link" data-filter=".tag3">Tag3</button>
   <button class="btn btn-link " data-filter=".tag4">Tag4</button>
</div>

 

Items

Notice the value in data-filter in Filters. If you want a portfolio item to respond to a filter button, put a css class on the item matching a button's data-filter value, but without the dot (.).

Example of an portfollio/facility item

 <figure class="col-12 col-lg-3 col-md-4 col-sm-6 tag4">
   <div class="item-wrap">
      <img src="image.jpg" alt="item1">
      <a href="" data-toggle="modal" data-target=".popup">
         <i class="icon-search fs-200"></i>
      </a>
    </div>
</figure>

* Popup modal could be used to display details of a portfolio item

Appear/Reveal animation

Wow js has been used to create appear/reveal anmations

Asset file used

  • wow.min.js
  • animate.css (for animation)

For this animaiton, Wow requires

  • class wow
  • another class from Animate.css (for example zoomIn)
  • attribute data-wow-delay
  • attribute data-wow-duration

Markup Example (at Service section's header)

<div class=".. wow zoomIn" data-wow-delay="0.3s" data-wow-duration="1.5s">
   <!-- text here-->          
</div>

 

Map & Contact Form

Google Map is in use here (asset file loaded from https://maps.googleapis.com).

You can change geolocation of the business, marker image on the map, business info displayed on the info window on the map and zoom level in init.js.

  1. var myGeoLocation = new google.maps.LatLng(-27.3051870,152.9888670);
  2. var mapMarker = 'images/marker.png';
  3. var businessName = 'Comapany Inc';
  4. var infoWindowText = '<h5>Comapany Inc</h5><p>370 Gympie Rd, Strathpine, QLD, 4500, Australia</p>';
  5. var zoomLevel = 16;

Lazy Load

JQuery Lazy Load plugin is used in this template to load some images, including some backgrounds. To make the plugin work, a "lazy" class is to be added to the tag and "data-src" attribute with the path to the image file.

An example of lazy load image

<img class="lazy" data-src="path/to/image.jpg">

To lazy load a background image

<div class="lazy" data-src="path/to/image.jpg">
   ....
</div>
